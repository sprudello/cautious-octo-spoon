## Installation Guide

- Clone this repository with:

```
git clone https://github.com/sprudello/cautious-octo-spoon.git
```
  
- Download XAMPP [here](https://www.apachefriends.org/download.html).

- Paste the cautious-octo-spoon folder into the xampp/htdocs folder.
- After downloading and installing XAMPP,lauch XAMPP.
- Start the Apache and the MySQL server in the control panel.
- Press on Admin in the control panel and move to phpMyAdmin.
- In the phpmyadmin panel, create the necessary database and the tables. The commands are in the database/tables.sql file.
- Go to localhost/cautious-octo-spoon and test it out.
